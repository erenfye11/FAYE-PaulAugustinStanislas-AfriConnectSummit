# AfriConnect Summit 2026

**Nom :** FAYE Paul Augustin Stanislas  
**Classe :** L1  CS
**Année :** 2026

## Description
Site vitrine complet d'une conférence tech fictive panafricaine nommée AfriConnect Summit. Un événement annuel qui réunit développeurs, entrepreneurs et investisseurs du continent africain.

## Technologies utilisées
- HTML5 sémantique
- CSS3 (Flexbox, Grid, variables CSS, animations)
- JavaScript Vanilla
- Bootstrap Icons
- Google Fonts (Syne + Inter)
- Git & GitHub

## Fonctionnalités JavaScript
- Dark mode avec localStorage
- Navbar dynamique au scroll
- Compte à rebours en temps réel
- Compteurs animés au scroll (IntersectionObserver)
- Onglets programme (Jour 1, 2, 3)
- Filtrage dynamique des intervenants
- Validation formulaire avec regex
- Bouton retour en haut
